## Stack
- ### Vite
- ### React 
- ### Redux 
- ### Typescript
- ### Electron 
- ### Vanilla Extract CSS

## Installation

Clone this repo and install all dependencies  
`yarn` or `npm install`

## Development

`yarn dev` or `npm run dev`

## Build

`yarn build` or `npm run build`

## Publish

`yarn dist` or `npm run dist`
